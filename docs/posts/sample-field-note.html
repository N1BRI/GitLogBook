Sample Field Note

A sample Markdown post showing how GitLogBook can publish station notes, activations, projects, and field reports.

This is a sample GitLogBook post. Replace it with your own station notes, activation reports, project logs, or field stories.

Posts live in `content/posts/` and are rebuilt into static pages with:

```sh
npm run posts
```

The public site shows recent posts in the About sidebar and provides a searchable post archive.
